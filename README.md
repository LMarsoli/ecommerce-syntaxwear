# SyntaxWear E-commerce Site

## Overview

SyntaxWear is an e-commerce website specializing in sneakers and athletic shoes. The site features a modern, responsive design with sections for product categories, featured products, and promotional banners. It is built using pure HTML and CSS, making it lightweight and easy to deploy.

## Features

- **Responsive Design**: Optimized for desktop and mobile devices.
- **Product Categories**: Sections for Casual, Sport, Modern, and Futuristic styles.
- **Hero Banner**: Promotional section highlighting featured products like the Krypton One.
- **Product Grid**: Showcase of various sneakers with images and descriptions.
- **Newsletter Signup**: Footer with email subscription form.
- **Social Media Links**: Integration with Instagram, WhatsApp, and TikTok.
- **Navigation**: Header with menu for Men's, Women's, and Outlet sections.

## Technologies Used

- **HTML5**: Structure and content
- **CSS3**: Styling and layout
- **Responsive Design**: Media queries for mobile compatibility

## How to Run

Since this is a static website, no server or build process is required. Simply open the `index.html` file in any modern web browser.

1. Clone or download the repository.
2. Navigate to the project directory.
3. Open `index.html` in your browser.
